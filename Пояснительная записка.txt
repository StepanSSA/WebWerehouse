Папки:
Описано на сайте в разделе информация
Классы:
IProductCRUDRepository = Интерфесы для рабосы с бд с действиями создания и обновления данных
IProductRepository = Интерфесы для рабосы с бд
AccountController = Контроллер для регистрации и входа на сайт
BossController = Контроллер для действий пользователя с ролью boss
Позволяет смотреть список пользователей, добавлять новых
ChemistryController = Контроллер для таблицы химия, доступен босу и кладовщику
Позволяет редактировать кол-во химии на складе, бос может добавлять новые данные
HomeController = Контроллер главной и информационной страниц
TableController = Контроллер для таблицы продукты, доступ имеют все роли
Позволяет вносить изменения в количество продукта на складе, 
общее кол-во продукта(произведено за все время), общее кол-во продаж этого продукта
Менеджер может убавлять и прибавлять кол-во продукта на складе
(его действия влияют только на колонку(всего проданно))
Бос и кладовщик делают тоже самое
(их действия влияют только на колонку (всего изготовленно))
ApplicationDbContext = Класс контекста данных
DataHelper = Класс для первоначальной записи данных в таблицу
Также добавляет новых пользователей и возвращает их список
MsSqlProductCRUDRepository и MsSqlProductRepository = Классы определяют интерфейсы
SimpleProductRepository = Простой репозиторий с пероначальными данными для таблиц продукты и химия
Product = Класс модели данных для продуктов
Сhemistry = Класс модели данных для химии
AddEmployee = Класс модели данных для новых сотрудников
AppUser = Класс модели для всех сотрудников
ErrorViewModel = Класс модели для ошибки
LoginViewModel = Класс модели для входа на сайт
RegisterViewModel = Класс модели для регистрации на сайте
UsersViewModel = Класс модели для отоброжения списка сотрудников
Представления:
Login = Вьюшка для входа на сайт
Register = Вьюшка для регистрации на сайте
Create в папке Boss = Всьюшка для регистрации сотрудника босом
List в папке Boss = Всьюшка для отображения таблицы сотрудников
В папке Chemistry
Create = Вьюшка для создания новой химии
Edit = Вьюшка для редактирования кол-ва химии на складе
TableChemistry = Всьшка для отображения таблыцы химии
В папке Home
Index = Вьюшка главной страницы
Privacy = Вьюшка иформации
В папке Shared
Все базовые Вьюшки
В папке Table
Create = Вьюшка для создания нового продукта
Edit = Вьюшка для Редактирования кол-ва продукта на складе
TableProducts = Таблица продуктов
